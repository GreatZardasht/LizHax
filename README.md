# Lizard Client
Lizard is a hacked client for Minecraft version 1.12.2, designed for anarchy servers such as 2b2t.org. Lizard looks to provide all the basic needs of an anarchy player as well as other handy features.

## Features:
- AutoMLG
- AutoSprint
- BHop
- CrystalAura
- EntityESP
- Fly
- KillAura
- Nofall
- Speed
